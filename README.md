# AI Security Agent

## Overview

AI Security Agent is a Python-based cybersecurity assistant that analyzes authentication logs, detects suspicious activities, calculates a security risk score, and generates professional incident reports using a locally hosted Large Language Model (Llama 3.2 via Ollama).

The project demonstrates how traditional Python programming and AI reasoning can be combined to build an intelligent security analysis system.


## Features

- Reads and parses server authentication logs
- Detects failed login attempts and suspicious IP addresses
- Identifies targeted user accounts
- Calculates a rule-based security risk score
- Classifies incidents by severity (Low, Medium, High, Critical)
- Generates AI-powered security incident reports using a local LLM
- Displays security alerts for high-risk incidents
- Recommends incident response actions
- Implements Human-in-the-Loop approval before executing actions
- Simulates common SOC response actions
- Saves incident reports automatically
- Interactive command-line interface


## Tech Stack

- Python
- Ollama
- Llama 3.2 (3B)
- VS Code

## Workflow

Read Logs → Parse Log Data → Detect Suspicious Activity → Calculate Risk Score → Generate AI Security Report → Display Security Alert → Recommend Response Actions → Human Approval → Simulate Security Response → Save Incident Report
